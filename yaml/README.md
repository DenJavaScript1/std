# deno-yaml
YAML parser / dumper for Deno

Heavily inspired from [js-yaml]

# Example
See `./example` folder and [js-yaml] repository.

# Limitations
- `binary` type is currently not stable
- `function`, `regexp`, and `undefined` type are currently not supported
